### Rodmguerra's ISS Studio 1.4-beta

- [Download](https://github.com/rodmguerra/issparser/releases/download/v1.4-beta/iss-studio-1.4-beta.zip) zip file, unzip it and run jar file (Required Java 8 or higher)
